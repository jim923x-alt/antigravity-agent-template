New Workspace Antigravity setup template
# Antigravity Reusable Agent Template v1.0

This template enforces:
- Optimal memory layering (Static → Long-term → Short-term)
- Token/cost savings
- Consistent best practices across all projects

**How to start any new project:**
1. Clone this repo into `.antigravity` folder in your new workspace
2. Type: `/bootstrap-template`
3. Agent will create all files automatically
4. Then tell it your project goal
